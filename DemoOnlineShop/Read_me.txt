This is a simple Automated UI test scenario for https://www.demoblaze.com/index.html DemoOnlineShop.

To run JavaScript version refer to DemoOnlineShop-JS folder.

To run Rapise UI Web test with RVL (Rapise Visual Language) refer to DemoOnlineShop folder.

In real life testing project tests can be expanded to cover all use case scenarios (pos&neg), all browsers, devices, resolutions.
And tests must be run as often as possible.
Defects are linked to every scenario. New tests are created according to new defects where needed/possible.
